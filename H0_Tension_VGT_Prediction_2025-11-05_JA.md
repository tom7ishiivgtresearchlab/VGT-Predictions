## 概要

H₀テンション問題—プランク衛星によるCMB観測から得られたハッブル定数（H₀ = 67.4 ± 0.5 km/s/Mpc）とSH0ESによる局所測定値（H₀ = 73.04 ± 1.04 km/s/Mpc）の間に存在する約5σの食い違い—は、標準ΛCDM宇宙論モデルにとって最も重大な課題の一つです。本文書は、仮想重力理論（VGT）が動的宇宙項 Λ(z) = Λ₀(1+z)^ε を通じてこのテンションの解決に寄与するという観測前予測を記録します。VGTの1ループ量子補正に基づき、ε ∼ 0.001–0.01 と予測され、これによりH₀テンションが約10–20%（ΔH₀ ≈ 0.6–1.1 km/s/Mpc）軽減されると期待されます。この予測は2027年11月に予定されるDESI-IIデータで検証可能です。重要なことに、VGTの意義はH₀テンション解決に留まらず、量子スケールから宇宙スケールまでを統一する理論的枠組みを提供し、宇宙項を第一原理から導出します。

## 1. 背景：H₀テンション問題

### 1.1 観測状況（2025年11月現在）

ハッブル定数 H₀ は宇宙の現在の膨張率を表します。二つの主要な測定方法は、有意に食い違う結果を示しています：

**初期宇宙（CMB）からの測定：**
- **Planck 2018:** H₀ = 67.4 ± 0.5 km/s/Mpc
- 方法：ΛCDMを仮定したCMB異方性からの推定

**後期宇宙（局所距離梯子）からの測定：**
- **SH0ES 2022:** H₀ = 73.04 ± 1.04 km/s/Mpc  
- 方法：セファイド変光星で較正されたIa型超新星

**食い違い：**
- 絶対差：5.64 km/s/Mpc
- 相対差：約8.4%
- 統計的有意性：**約5σ**（偶然の確率 < 0.00006%）

### 1.2 重要性

このテンションは現代宇宙論に対する根本的な挑戦を表しています：

1. **頑健性：** 両測定は複数の独立したチームによって精査・確認されている
2. **系統誤差の検証：** 複数の検証研究が明白な系統誤差を除外している
3. **持続性：** テンションは過去10年間、データの改善とともに持続・強化されている
4. **含意：** 標準ΛCDMモデルを超える物理学を示唆する可能性

### 1.3 現在の理論的状況

提案されている解決策には以下が含まれます：
- 初期ダークエネルギーモデル
- 修正重力理論
- 基礎定数の変動
- エキゾチックニュートリノ物理学
- **動的宇宙項（VGTアプローチ）**

しかし、コンセンサスを得た解決策はなく、ほとんどが全ての宇宙論的データを同時に説明することに困難を抱えています。

## 2. 理論的枠組み：VGTアプローチ

### 2.1 ΛCDMの限界

標準ΛCDMモデルは以下を仮定します：
```
Λ = 定数
```

**問題点：**
1. 特定の値に対する理論的正当化がない
2. 「宇宙定数問題」（量子場理論との120桁の食い違い）
3. H₀テンションを説明できない
4. 量子重力との接続がない

### 2.2 VGTの提案：動的宇宙項

仮想重力理論は以下を提案します：

```
Λ(z) = Λ₀(1+z)^ε
```

ここで：
- **z**: 宇宙論的赤方偏移
- **Λ₀**: 現在の宇宙項の値
- **ε**: 時間発展パラメータ

**重要な区別：** ε は理論から導出されるものであり、データに合わせた自由パラメータではありません。

### 2.3 ε の理論的導出

VGT Volume 1（Phases 1-20）から、量子場理論（QFT）と一般相対性理論（GR）を統合して：

**1ループ量子補正が予測：**
```
ε ∼ α/(4π) ≈ 0.001 - 0.01
```

ここで α は関連する結合定数です。

**物理的解釈：**
- 宇宙項は宇宙時間とともに動的に進化する
- 進化率は量子補正によって決定される
- 自然な理論的予測（恣意的ではない）

### 2.4 H₀テンション軽減のメカニズム

**因果連鎖：**

```
動的 Λ(z)
    ↓
高赤方偏移でのハッブルパラメータ H(z) の修正
    ↓
CMB（z ≈ 1090）までの角径距離の変化
    ↓
CMB音響スケールから推定される H₀ の変化
    ↓
局所測定との食い違いの軽減
```

**定量的見積もり：**
- ε = 0.01 の場合：予想される軽減 ΔH₀ ≈ 0.6-1.1 km/s/Mpc
- これは全テンションの10-20%を表す
- 残存テンションには追加の物理が必要な可能性

**もし ε が有意に大きい場合（>0.05）：**

**もし ε ≈ 0 の場合（ΛCDMからのずれなし）：**
- 現在の赤方偏移範囲ではVGT予測が検証されない
- しかし、より高赤方偏移（z > 5）で差異が現れる可能性

### A.2 角径距離

CMBまでの角径距離：

```
Dₐ(z) = (c/H₀) × (1/(1+z)) × ∫₀^z dz'/√[Ωₘ(1+z')³ + ΩΛ(1+z')^ε]
```

ε > 0 の場合、この積分はΛCDMとは異なる結果を与え、推定されるH₀を修正します。

### A.3 音響地平線スケール

CMB音響スケール：

```
θ* = rs/Dₐ(z*)
```

ここで rs ≈ 147 Mpc は再結合時（z* ≈ 1090）の音響地平線です。

もし Dₐ が動的Λによって修正されれば、観測された θ* から推定される H₀ もそれに応じて変化します。

### A.4 予想される効果の大きさ

ε = 0.01 の場合：
```
ΔH₀/H₀ ≈ (ε/2) × ln(1+z*) ≈ 0.01 × 3.5 ≈ 3.5%

絶対変化：ΔH₀ ≈ 0.7 km/s/Mpc
```

これは粗い見積もりです；精密な計算には数値積分が必要です。
